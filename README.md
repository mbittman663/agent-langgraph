Making a chained agentic solution using LangGraph and OpenAI

///// Prerequisites /////
- Will need to make your own .env file with your OpenAI API Key
- Will need to install packages from requirements.txt

///// To Run with Streamlit UI /////
Open Terminal
py -m streamlit app.py (should open tab in your browser to interact with agents)

///// To Run without Streamlit UI - run & use in terminal /////
Open Terminal 
py -m main.py (can interact in the terminal)
