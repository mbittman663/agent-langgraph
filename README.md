**Making a chained agentic solution using LangGraph and OpenAI (Python 3.14.3)**

///// Prerequisites /////
- Will need to make your own .env file with your OpenAI API Key
- Set up & activate virtual environment (venv)
- Will need to install packages from requirements.txt (after setting up venv if desired)
  - pip install -r requirements.txt (to install the necessarily libraries in one batch)

///// (Optional) Set Up & Activate Virtual Environment (venv) /////
- py -m venv venv (to create for the first time)
- .\venv\Scripts\Activate.ps1 (to activate)

///// To Run with Streamlit UI /////
- Open Terminal
- py -m streamlit run app.py (should open tab in your browser to interact with agents)

///// To Run without Streamlit UI - run & use in terminal /////
- Open Terminal 
- py -m main.py (can interact in the terminal)
